## Unity 에디터 설정 체크리스트

```
[ ] 각 Step 씬에 StepController 오브젝트 배치
    - Dialogue Sequence 배열에 해당 DialogueData asset 연결
    - Next Scene 필드에 다음 씬 이름 입력

[ ] 각 씬에 DialogueManager, ChoiceSystem, DialogueUI, ChoiceUI 배치
    - Prefab으로 만들어 재사용 권장

[ ] URP Global Volume 설정
    - Vignette 추가 (Intensity: 0.2 시작)
    - Chromatic Aberration 추가
    - Color Adjustments 추가 (Saturation: 0 시작)

[ ] AudioMixer "BGM_Pitch", "BGM_Distortion" exposed parameter 설정

[ ] 도윤 정면 응시 스프라이트
    - 측면 → 3/4 → 정면 순서 4~6프레임 제작
    - FourthWallBreak.stareFrames 배열에 연결

[ ] 도윤 스프라이트 Sorting Layer "Foreground" 설정
    (흑백 처리 시 도윤만 컬러 유지되도록 URP Render Feature로 분리)

[ ] Build Settings에 씬 10개 모두 추가 (순서 중요)
```
